Quick Start

The DailyAI API is completely open. No authentication or API keys are required.

1. Fetch the AI news feed

curl -X GET 'https://dailyai-news.onrender.com/api/v1/feed'

2. Python example

import requests

# Get latest AI news
response = requests.get(
    "https://dailyai-news.onrender.com/api/v1/feed",
    params={"topic": "all", "language": "en", "limit": 10}
)
articles = response.json().get("articles", [])

for article in articles:
    print(f"📰 {article['headline']}")
    print(f"   Source: {article['source_name']}")
    print()

3. JavaScript example

const response = await fetch(
  'https://dailyai-news.onrender.com/api/v1/feed?topic=all&limit=10'
);
const { articles } = await response.json();

articles.forEach(a => console.log(`📰 ${a.headline} (${a.source_name})`));

API Endpoints

GET /api/v1/feed

Get the curated AI news feed with filtering and pagination.

ParameterTypeDefaultDescription
topicstringallFilter by topic: all, For You, AI Models, Research, Business, Tools, Top Stories, Tech & Science
countrystringGLOBALCountry code: GLOBAL, US, GB, DE, IN
languagestringenLanguage: en, de
offsetint0Pagination offset
limitint15Items per page (max 30)
GET /api/v1/trending

Get trending story threads — groups of related articles about the same event.

ParameterTypeDefaultDescription
countrystringGLOBALCountry code
languagestringenLanguage
GET /api/v1/categories

Get the list of available categories and their item counts for a specific feed configuration.

ParameterTypeDefaultDescription
countrystringGLOBALCountry code
languagestringenLanguage

Authentication

DailyAI is currently an open-source non-commercial project. No API keys or Authorization headers are needed to query the public /api/v1/ endpoints.

Rate Limits

While the API is open, please respect standard rate limits. Automated circumvention or aggressive polling may result in temporary IP blocks to preserve resources. If you need dedicated access, consider self-hosting the project via the GitHub repo.
